## Suporte a JavaScript

O Biome suporta a versão ES2023 do JavaScript.

O Biome suporta apenas a síntaxe oficial. O time inicia o desenvolvimento de uma nova síntaxe quando uma proposta chega no [estágio 3](https://github.com/tc39/proposals#stage-3).

## Suporte a TypeScript

O Biome suporta a versão 5.2 do TypeScript.

## Suporte a super linguagens no HTML

Essas linguagens requerem suporte adequado para a análise sintática de CSS, HTML e JavaScript. Uma vez que esses analisadores sintáticos estejam disponíveis, será possível começar a trabalhar com essas super linguagens.
