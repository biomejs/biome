<div class="hero-code">
    <h2 class="head">Resolver problemas, aprender las mejores prácticas</h2>

    **Biome es un [linter eficiente](https://github.com/biomejs/biome/tree/main/benchmark#linting)** para _JavaScript_, _TypeScript_ y _JSX_ que incluye **[más de 190 reglas](https://biomejs.dev/linter/rules/)** de ESLint, TypeScript ESLint y [otras fuentes](https://github.com/biomejs/biome/discussions/3).
    
    ¡**Biome genera diagnósticos detallados y contextualizados** que te ayudan a mejorar tu código y a convertirte en un mejor programador!

    <LinterExample />

    Prueba el linter de Biome en el [espacio de pruebas](/playground/) o directamente en tu proyecto:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome lint --apply ./src
    ```

</div>


<div class="hero-code">
    <h2 class="head">Todo al mismo tiempo</h2>

    No solo puedes formatear y revisar tu código por separado, ¡sino que puedes hacerlo **todo de una vez con un solo comando**!

    Cada herramienta se integra perfectamente con las demás para crear **una cadena de herramientas cohesionada** para proyectos web.

    Ejecuta todas las herramientas con el comando `check`:

    ```shell
    npm i -D --save-exact @biomejs/biome
    npx @biomejs/biome check --apply ./src
    ```

</div>
